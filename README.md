# Bookstore Backend
A bookstore backend using Nest.js microservices. The services communicate with TCP connections

## Project setup

```bash
$ git clone https://github.com/goutamthakur/bookstore-backend.git
```


```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
